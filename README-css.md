# CSS Review
This is my review of week 1 in CSS.

## Examples of CSS selectors

#### Class Selector

```css
.class-selector{
    background-color: white;
}
```

#### ID Selector

```css
#id-selector{
    background-color: green;
}
```

#### Select an HTML Tag

Just put the name of the html tag without anything in front

```css
body {
    background-color: purple;
}
```
